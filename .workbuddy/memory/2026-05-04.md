- StanceSelector 渲染 `0/undefined`

### 问题3: baseURL 硬编码 `localhost:3000`，Mock 走「先网络失败再回退」路径
- 每次 API 请求先报 `ERR_CONNECTION_REFUSED`，再由 response.error 拦截器转到 Mock
- 某些请求的 config.url 包含完整 URL（`http://localhost:3000/api/xxx`），路径匹配失效
- **修复**: 重写 `request.js`：请求拦截器直接 `Promise.reject({__MOCK__: true})` → 响应拦截器识别后走 `handleMockRequest`；URL 路径统一用 `getRelativePath()` 提取；`baseURL` 改为 `/api`
