# 端到端联调测试文档

## 测试流程
注册 → 用户名校验 → 逻辑测试 → AI辩论考核 → 审核通过 → 进入首页

## 服务状态
- ✅ 后端服务：http://localhost:5000
- ✅ 前端服务：http://localhost:3000

## 测试步骤

### 1. 注册账号
访问：http://localhost:3000/register

**测试数据：**
- 账号类型：手机号
- 手机号：13800138000
- 用户名：张三（必须包含真实姓氏）
- 密码：123456
- 确认密码：123456
- 验证码：1234（已预设）
- 勾选用户协议

**预期结果：**
- 注册成功后自动跳转到逻辑测试页面
- URL：http://localhost:3000/check/logic?userId=X

### 2. 逻辑测试
**操作：**
- 完成5道逻辑题
- 需要答对3题以上（60分）才能通过

**预期结果：**
- 显示得分和答对题数
- 点击"下一步"跳转到AI辩论考核

### 3. AI辩论考核
**操作：**
- 系统随机分配一个辩题
- 输入至少50字的观点
- 建议使用逻辑连接词：因为、所以、首先、其次、例如等
- 保持理性表达，避免过多感叹号和绝对化词汇

**评分标准：**
- 逻辑性评分（50分基础）：
  - 使用因果词 +15分
  - 使用序列词 +15分
  - 使用举例词 +20分
- 理性度评分（50分基础）：
  - 过多标点 -20分
  - 绝对化词汇 -10分
  - 使用可能性词汇 +20分
  - 使用转折词 +20分

**预期结果：**
- 显示逻辑性、理性度和综合得分
- 需要60分以上才能通过
- 点击"查看审核结果"

### 4. 审核结果
**预期结果：**
- 如果逻辑测试≥60分 且 辩论考核≥60分：
  - 显示"审核通过"
  - 账号状态更新为激活（status=1）
  - 可以点击"进入首页"
- 如果未通过：
  - 显示失败原因
  - 24小时后可重试
  - 累计3次失败将限制30天

### 5. 进入首页
**预期结果：**
- 成功进入首页
- 显示"逆熵 - 理性辩论平台"

```bash

# 提交答案

{
  "userId": 1,
  "answers": [
    {"question": "题目1", "answer": 0},
    {"question": "题目2", "answer": 1}

```

### 辩论考核接口
```bash
# 获取辩题
GET http://localhost:5000/api/check/debate-topic

# 提交观点
POST http://localhost:5000/api/check/debate
Content-Type: application/json

{
  "userId": 1,
  "topic": "远程办公是否应该成为主流工作模式",
  "speech": "我认为远程办公应该成为主流工作模式。首先，远程办公可以节省通勤时间，提高工作效率。其次，它能够降低企业的运营成本。例如，研究表明远程办公可以减少30%的办公场地费用。"
}
```

### 查询审核结果
```bash
GET http://localhost:5000/api/check/result/:userId
```
