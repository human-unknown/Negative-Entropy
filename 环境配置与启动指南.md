## 🖥️ 系统要求

### 必需软件
- **Node.js**: >= 18.0
- **MySQL**: >= 8.0
- **npm**: >= 9.0

### 推荐配置
- **操作系统**: Windows 10/11, macOS 10.15+, Ubuntu 20.04+
- **内存**: >= 4GB
- **磁盘空间**: >= 2GB

## ⚙️ 环境配置

### 1. 克隆项目（如适用）

```bash
git clone <repository-url>
cd 逆熵
```

### 2. 后端环境配置

#### 安装依赖
```bash
cd backend
npm install
```

#### 配置环境变量

创建 `.env` 文件（复制 `.env.example`）：

```bash
# Windows
copy .env.example .env

# macOS/Linux
cp .env.example .env
```

编辑 `.env` 文件：

```env
# 服务端口
PORT=3000

# 数据库配置
DB_HOST=localhost
DB_USER=root
DB_PASSWORD=your_mysql_password
DB_NAME=debate_platform

# JWT密钥（生产环境请修改）
JWT_SECRET=your-secret-key-change-in-production
```

**重要提示**：
- `DB_PASSWORD`: 填写你的MySQL root密码
- `JWT_SECRET`: 生产环境必须修改为复杂密钥

### 3. 前端环境配置

#### 安装依赖
```bash
cd frontend
npm install
```

#### 配置API地址

前端默认连接 `http://localhost:3000`，如需修改请编辑 [`frontend/src/api/request.js`](frontend/src/api/request.js)：

```javascript
const request = axios.create({
  baseURL: 'http://localhost:3000/api',  // 修改为实际后端地址
  timeout: 10000
});
```

## 🚀 启动命令

### 方式1：分别启动（推荐用于开发）

#### 启动后端
```bash
# 进入后端目录
cd backend

# 启动服务
node server.js

# 或使用开发模式（如已配置nodemon）
npm run dev
```

**预期输出**：
```
Server running on port 3000
Database connected successfully
```

#### 启动前端
```bash
# 新开一个终端，进入前端目录
cd frontend

# 启动开发服务器
npm run dev
```

**预期输出**：
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:5173/
➜  Network: use --host to expose
```

### 方式2：使用并发启动（可选）

如需同时启动前后端，可安装 `concurrently`：

```bash
# 在项目根目录
npm install -g concurrently

# 创建启动脚本
concurrently "cd backend && node server.js" "cd frontend && npm run dev"
```

### 访问应用

- **前端地址**: http://localhost:5173
- **后端API**: http://localhost:3000/api
- **API文档**: 暂无（可使用Postman测试）

## ❓ 常见问题

### 1. 数据库连接失败

**错误信息**：`Error: connect ECONNREFUSED 127.0.0.1:3306`

**解决方法**：
- 检查MySQL服务是否启动
- 检查 `.env` 中的数据库配置是否正确
- 检查MySQL端口是否为3306

```bash
# Windows - 检查MySQL服务
net start | findstr MySQL

# macOS/Linux - 检查MySQL服务
ps aux | grep mysql
```

### 2. 端口被占用

**错误信息**：`Error: listen EADDRINUSE: address already in use :::3000`

**解决方法**：
```bash
# Windows - 查找并结束占用进程
netstat -ano | findstr :3000
taskkill /PID <PID> /F

# macOS/Linux - 查找并结束占用进程
lsof -i :3000
kill -9 <PID>

# 或修改端口
# 编辑 backend/.env，修改 PORT=3001
```

### 3. 前端无法连接后端

**错误信息**：`Network Error` 或 `CORS Error`

**解决方法**：
- 确认后端已启动（http://localhost:3000）
- 检查前端API配置（frontend/src/api/request.js）
- 检查后端CORS配置（backend/src/app.js）

### 4. SQL脚本执行失败

**错误信息**：`ERROR 1050: Table already exists`

**解决方法**：
```sql
-- 删除数据库重新创建
DROP DATABASE debate_platform;
CREATE DATABASE debate_platform CHARACTER SET utf8mb4 COLLATE utf8mb4_unicode_ci;

-- 重新执行SQL脚本
```

### 5. 注册时验证码错误

**说明**：当前为测试模式，验证码固定为 `1234`

如需修改，编辑 [`backend/src/controllers/authController.js`](backend/src/controllers/authController.js)

### 6. JWT Token过期

**错误信息**：`401 Unauthorized`

**解决方法**：
- 重新登录获取新Token
- 或修改Token过期时间（backend/src/middlewares/auth.js）

### 7. 前端页面空白

**解决方法**：
- 检查浏览器控制台错误信息
- 清除浏览器缓存
- 检查前端是否正确启动（http://localhost:5173）
